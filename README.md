# ⏱ Digital Stopwatch (Java Project)

## 📌 Project Description
This project is a console-based Digital Stopwatch developed using Java and Object-Oriented Programming (OOP) concepts.  
It allows users to measure elapsed time using start, stop, reset, and display options through a simple menu-driven interface.

---

## ❓ Why This Project?
This project is created to:
- Understand the practical use of Java programming  
- Learn and apply Object-Oriented Programming (OOP) concepts  
- Work with real-time functions like System.currentTimeMillis()  
- Build an interactive console-based application  

---

## 🚀 Features
- Start the stopwatch  
- Stop the stopwatch  
- Reset the stopwatch  
- Display elapsed time in milliseconds  
- Menu-driven interface  

---

## 🛠 Technologies Used
- Java  
- OOP Concepts (Encapsulation, Classes & Objects)  

---

## ⚙ How It Works
- Uses System.currentTimeMillis() to track time  
- Calculates elapsed time when stopped  
- Displays time when selected from menu  

---

## ▶ How to Run

Compile:
javac Main.java

Run:
java Main

---

## 📋 Menu Options
1 → Start  
2 → Stop  
3 → Reset  
4 → Show Time  
5 → Exit  

---

## 📂 Project Structure
- Stopwatch class → logic  
- Main class → user interaction
